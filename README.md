Pair It Right – Memory Game
Description:
A fun and interactive memory matching game built with HTML, CSS, and JavaScript. Flip cards to find matching pairs as fast as you can! Features a timer, shuffle logic, responsive design, and a “You Won” popup.
Features:
Responsive design for mobile and desktop
Timer starts on first click and stops when all pairs are matched
Cards shuffle automatically at the start
Smooth card flip animations
“You Won” popup with time display
Play Again button
How to Play:
Click a card to flip it.
Find its matching pair.
Repeat until all cards are matched.
Timer stops and a popup appears showing your time.
Technologies Used:
HTML
CSS (Flex/Grid, transitions)
JavaScript (DOM manipulation, event listeners)
